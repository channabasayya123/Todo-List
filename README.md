 Features

 Add new tasks instantly

 Mark tasks as completed

 Delete tasks you don’t need

 Responsive design (works on mobile, tablet, and desktop)

Simple and clean UI

 Lightweight — built with pure JavaScript (no frameworks)


Technologies Used

HTML5 – Structure of the app

CSS3 – Styling and layout

JavaScript (ES6) – Interactivity and task management logic


 Purpose

This To-Do List App is designed to help users stay organized by tracking:

Student study tasks

Daily household activities

Work-related reminders

 Personal goals and routines


 How to Use

1. Type your task into the input box

2. Click the Add button

3. Mark tasks as Completed when done

4. Delete tasks using the trash icon / delete button
